# Support

Thanks for using LoTaR.

## Questions / Help

Start with the docs:

- Docs index: docs/README.md
- Help index: docs/help/index.md
- Error model and common failures: docs/help/errors.md
- Web server + API reference: docs/help/serve.md

If you’re still stuck, open an issue:

- https://github.com/localtaskrepo/lotar/issues

## Bug Reports

Please open a GitHub issue and include:

- What you expected vs what happened
- Repro steps (ideally in a fresh temp repo)
- Output of `lotar --version`
- OS + shell (and Docker version if applicable)
- Any relevant logs (`--verbose` helps)

## Feature Requests

Open an issue describing:

- The workflow you’re trying to improve
- Why LoTaR’s current behavior isn’t enough
- Any constraints (repo scale, monorepo, CI usage)

## Security

Please don’t open public issues for security-sensitive reports.

- Preferred: GitHub security advisories: https://github.com/localtaskrepo/lotar/security/advisories/new
